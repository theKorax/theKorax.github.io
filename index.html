Projects

Online Learning Platform integrated with Automated Subjective Answer Grading

This project served as my undergraduate thesis and was undertaken to investigate the limits of Machine Learning and Natural Language Processing in evaluating non-objective human input. It integrates the automated grading system into a demo online learning platform, with test questions and instructor-provided answers which the grading system compares against the "student"-provided input.

• Languages & Tools: Python, MySQL, Jupyter Notebook, React, Git version control.
• Core Functionality: The platform employs AI and NLP techniques, such as text classification with LinearSVC, semantic similarity using BERT model, document embedding using Doc2Vec, data preprocessing, tokenization, feature extraction, data cleaning, and visualization.
• Evaluation Metrics: Precision, recall, and F1-score to assess model performance.
• User Interaction: Instructors can create courses, create tests, set test questions and the expected correct answer. Students can view the course catalogue and enroll in courses, view their tests and answer the questions.

Results: The project made strides towards bridging the gap in advanced subjective evaluation on online learning platforms.

Dog Image Classifier

Completed a project that builds an end-to-end deep learning multiclass image classifier to accurately identify the breed of a dog in a given image.

• Languages & Tools: Jupyter Notebook, Python, pandas, TensorFlow, mobilenetv2 model, Git version control.
• Core Functionality: Processed and analyzed unstructured visual data from kaggle's dog breed identification competition dataset to predict the breed of a dog. Utilized pandas for data manipulation and visualization
• Communication: Documented and presented findings in a structured Jupyter Notebook markdown format, detailing the model's functionality and its pipeline process.

Results: Developed a model that accurately identifies to a high degree of accuracy, the breed of a dog in a given image.

Results: Gained hands-on experience in the full game development pipeline process, from gameplay programming, to asset creation and animation, to level design and audio engineering, and finally creating the mobile version for android devices.

Word Flip Game

A word association game where players begin a word by starting it with the last letter of the previous word. The game takes some inspiration from freestyle games such as "Pick up and kill it" on the popular MTV show, which pushes players to think fast on their feet, and type even faster.

• Languages & Tools: C#, Unity.
• Core Functionality: I used the official Scrabble English words dataset. I worked on the search algorithm to retrieve the player input from the dataset if the word exists and implemented data structures to keep track of all valid words provided in the current game round.

Results: Delivered a comprehensive, web-based casual game that built on core aspects of computer science.

Loan Repayment App Model

This project aims to develop a loan repayment calculator app to model a real life loan system, including compounding interest rates, flexible repayment structures, bank(borrower) statistics and predictions of the earnings of x months ahead.

• Languages & Tools: Python, Jupyter Notebook, Scikit-Learn, Git version control.
• Core Functionality: Add a user, request a loan, make payment, Show company's statistics (using a linear regression model)
• User Interaction: Users can input data choices through a text-based interface and model how a real life loan repayment system would function.

Results: The system accurately models loan repayment, and goes some way towards making predictions of the earnings of x months ahead.
